Hexo 常用命令行

发表于 2017-01-15 | 分类于 Hexo |

以下所有的命令都是在hexo目录下进行
部分常用的命令，待补充

新建文章

hexo n “article name”
hexo new “article name”
hexo new post “article name”
hexo new post articleName

生成编译

hexo g
hexo generate

部署

hexo d
hexo deploy

在部署前先生成

以上两个可以直接写出以下形式

hexo d -g

本地debug调试

hexo s -debug
hexo server -debug

清除缓存

hexo clean
