Android 录制视频然后转GIF图

发表于 2017-01-17 |

最近使用Hexo搭建个人博客，开始着手整理下自己写的代码，Demo展示静态图不好体现，现在使用Android命令行进行录制视频，然后通过其他工具转为GIF，更为生动。

注:Android4.4(API19)以上的手机，命令screenrecord可以帮助我们简单实现。

录制命令

adb shell screenrecord /sdcard/videoname.mp4

/sdcard/videoname.mp4 是视频所保存的目录和名称。
输入命令按Enter后，就会开始录制180s，Ctrl+C可以提前终止。

获取视频文件

adb shell pull /sdcard/videoname.mp4

/sdcard/videoname.mp4 就是我们刚才录制存放的目录地址，此命令将SD卡中的视频文件保存到本地。

注:命令所在的目录下

视频转GIF

目前使用EZGIF 和 IMGFLIP还是相当便捷的，也有一些其他的工具，可以自己查阅。
